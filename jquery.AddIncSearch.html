Test page for AddIncSearch

The selectbox below contains 1600 filenames found on a unix /usr/include filesystem.

Try the following things:

  • type 'tiff' to see all includes for tiff.
  • type 'silly' to see what happens when there is no match.
  • scroll to the bottom of the list to see that only part of the available files are listed for performance.
  • delete the selected word to see the start of the list
